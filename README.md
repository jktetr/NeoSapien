# NeoSapien static site v4

Open `index.html` directly, or run:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## GitHub Pages

1. Create a new GitHub repo.
2. Upload all files in this folder, including the `assets/` directory.
3. Go to Settings -> Pages.
4. Source: Deploy from branch.
5. Branch: `main`, folder: `/root`.
6. Save.

Videos work on GitHub Pages as long as the `.mp4` files are committed into the repo and referenced with relative paths, as done here.
